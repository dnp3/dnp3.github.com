Welcome to opendnp3

This is a commerically supported, rigorously tested, open source implementation of IEEE-1815 (dnp3). The core library is C++ that can target many server and embedded platforms. Java and .NET bindings are provided.

Opendnp3 is not endorsed by dnp.org or IEEE. It is a seperate project that tracks the standard with a reference implementation. Automatak, LLC is a premium member of dnp.org and participates on the dnp technical committee.

How long has this library been around?

It's a commercial fork of this project. The library has been in use since 2006 and has been deployed extensively across the industry since the open source release in 2010. This fork is maintained and supported by Automatak, LLC. We have modified the library as described below. These improvements are based on almost 7 years of user feedback and real world deployment experience.

Why would I use this over the original project?

Many engineering hours have gone into improving the original codebase. Here's a short list:

  • Defect-free custom Java bindings
  • Mutli-core support improves responsiveness and scalability
  • Extensive online documentation
  • Object model for managing channels, masters, and outstations
  • Masters and outstations independently handle all control functions and types
  • There is only one library with one namespace: opendnp3
  • Updated build systems (GNU autotools and Visual Studio)
  • Size reduction for embedded systems
  • Aggressive release cycle and compliance maintainence
