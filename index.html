Welcome

This is a commerically supported OSS implementation of IEEE-1815 (dnp3) sponsored by Automatak and provided under the commercially-friendly Apache License.

The core library is modern C++11 that can target many server and embedded platforms. Java and .NET bindings are provided as well.

Checkout the code on Github. Join our Google Group and introduce yourself. The community is here to help.

Features

  • Free stack and tools
  • Excellent unit test coverage
  • Achieve DNP3 Level 2+ conformance with ease
  • Written in modern C++11 and designed to target an OS
  • Non-blocking, event-driven architecture that scales to OS socket limits
  • Multi-core support for large frontends and device simulations
  • Simple, consistent programming model that lowers time to market

Quality

  • Proven solution with many production deployments
  • Unit test coverage in excess of 90%
  • Industry leading security and robustness
  • Type-safe API that is both safe and easy to use
