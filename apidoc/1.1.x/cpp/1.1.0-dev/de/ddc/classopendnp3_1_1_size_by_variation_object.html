Detailed Description

Represents a byte stream object whose size is encoded by its variation.

As per IEEE 1815 "Standard for Electric Power Systems Communications - Distributed Network Protocol (DNP3)", Annex A "DNP3 Object Library", page 429, the only DNP3 object types that support the "size-by-variation" feature are 100 (floating-point), 110 (octet string), 111 (octet string event), 112 (virtual terminal output block), and 113 (virtual terminal event data).
