Detailed Description

The root class for all dnp3 applications.

Used to retrieve communication channels on which masters / outstations can be bound. The stack is event-based and driven by a thread pool. All callbacks come from a thread in the pool.

    DNP3Manager manager(std::thread::hardware_concurrency());
