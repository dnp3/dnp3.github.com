Detailed Description

Base class shared by all of the DataPoint types.

There are 5 major data types and they all have a value, timestamp and a quality field. The timestamp should reflect when the value was measured or calculated. The quality field should be set approriately depending on the data type. Each datatype has a its own defintion of the quality field that indicate specific conditions but all of the datatypes define an XX_ONLINE bit, that is the default "nominal" value. This quality field is not for applying alarming information, that needs to be done with Binaries or in other channels.
